## 시스템 개요

CHEMON 견적관리 시스템은 비임상 시험 서비스의 견적서 작성, 고객 관리, 계약 관리를 위한 통합 플랫폼입니다.

### 주요 기능

- **독성시험 견적서**: GLP/Non-GLP 독성시험 견적 작성
- **효력시험 견적서**: 효력시험 모델 기반 견적 작성
- **임상병리검사 견적서**: 임상병리 검사항목 기반 견적 작성
- **고객사 관리**: 고객사 정보 및 담당자 관리
- **계약/시험 관리**: 계약서 작성 및 시험 진행 관리
- **리드/파이프라인**: 영업 기회 추적 및 관리

## 로그인

### 접속 방법

1. 웹 브라우저에서 시스템 URL 접속
2. 이메일과 비밀번호 입력
3. "로그인" 버튼 클릭

- 이메일: `admin@chemon.co.kr`
- 비밀번호: `admin1234!`
- ⚠️ **첫 로그인 후 반드시 비밀번호를 변경하세요!**

## 영업관리

### 리드 관리

신규 영업 기회를 등록하고 관리합니다.

**리드 등록 방법:**

1. 영업관리 → 리드 관리 메뉴 클릭
2. "새 리드" 버튼 클릭
3. 고객 정보, 문의 내용 입력
4. 저장

### 파이프라인

영업 진행 단계를 칸반 보드 형태로 관리합니다.

**단계:**

1. 문의접수
2. 검토
3. 견적서 송부
4. 연구소 현황 파악
5. 시험의뢰검토
6. 계약진행
7. 시험접수
8. 관리

### 고객사 관리

고객사 정보를 등록하고 관리합니다.

**고객사 등록:**

1. 영업관리 → 고객사 관리 메뉴 클릭
2. "새 고객사" 버튼 클릭
3. 회사명, 담당자, 연락처 등 입력
4. 저장

## 견적관리

### 견적서 작성

#### 독성시험 견적서

1. 견적관리 → 견적서 작성 클릭
2. "독성시험" 유형 선택
3. 단계별 정보 입력:
   - **기본정보**: 고객사, 프로젝트명, 모달리티
   - **시험조건**: 동물 종, 투여경로, 시험기간 등
   - **검사항목**: 필요한 검사 선택
   - **금액확인**: 할인 적용 및 최종 금액 확인
4. 저장 또는 PDF 다운로드

#### 효력시험 견적서

1. 견적관리 → 견적서 작성 클릭
2. "효력시험" 유형 선택
3. 단계별 정보 입력:
   - **기본정보**: 고객사, 프로젝트명
   - **모델선택**: 효력시험 모델 선택
   - **항목설정**: 시험 항목 및 수량 설정
   - **금액확인**: 최종 금액 확인
4. 저장

#### 임상병리검사 견적서

1. 견적관리 → 견적서 작성 클릭
2. "임상병리" 유형 선택
3. 단계별 정보 입력:
   - **기본정보**: 의뢰기관, 담당자, 시험기준(GLP/Non-GLP)
   - **검체정보**: 동물 종, 검체 종류, 검체 수
   - **검사항목**: CBC, 생화학, 응고검사 등 선택
   - **금액확인**: QC 비용, 할인 적용, 최종 금액
4. 저장

### 견적 목록

- 작성된 모든 견적서 조회
- 유형별 필터 (독성/효력/임상병리)
- 상태별 필터 (작성중/제출/수주/실주)
- 검색 기능

### 패키지 템플릿

자주 사용하는 검사 조합을 패키지로 저장하여 재사용할 수 있습니다.

## 계약/시험

### 계약 관리

수주된 견적서를 계약서로 전환하고 관리합니다.

**계약서 작성:**

1. 계약/시험 → 계약 관리 메뉴 클릭
2. "새 계약" 버튼 클릭 또는 견적서에서 전환
3. 계약 조건 입력
4. PDF 다운로드 가능

### 시험 관리

진행 중인 시험을 관리합니다.

### 시험의뢰서 (임상병리)

임상병리검사 시험의뢰서를 관리합니다.

**시험의뢰서 생성:**

1. 승인된 임상병리 견적서에서 "시험의뢰서 전환" 클릭
2. 추가 정보 입력 (보고서 유형, 검체 처리 방법 등)
3. 제출

**상태 관리:**

- 작성중 → 제출됨 → 접수완료 → 진행중 → 완료

### 상담기록

고객 상담 내용을 기록하고 관리합니다.

## 자주 묻는 질문 (FAQ)

### Q: 견적서 PDF가 다운로드되지 않아요

A: 브라우저의 팝업 차단을 해제해주세요.

### Q: 고객사가 목록에 안 보여요

A: 고객사 관리에서 먼저 고객사를 등록해야 합니다.

### Q: 임상병리 검사항목이 안 보여요

A: 관리자에게 문의하여 마스터데이터 시드를 요청하세요.

### Q: 비밀번호를 잊어버렸어요

A: 관리자에게 비밀번호 초기화를 요청하세요.
