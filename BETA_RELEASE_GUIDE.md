## 시스템 개요

CHEMON CRM은 비임상 시험 견적서 작성부터 계약, 시험 관리까지의 전체 업무 프로세스를 지원하는 통합 관리 시스템입니다.

- **프론트엔드**: Vercel 배포 (Next.js 14)
- **백엔드**: Render 배포 (Express.js + PostgreSQL)
- **다중 사용자 지원**: 모든 데이터는 서버에 저장되어 팀원 간 공유

## 핵심 워크플로우

**리드 → 견적 → 계약 → 시험** 전체 흐름이 연결되어 있습니다.

### 1. 리드 관리 (`/leads`, `/pipeline`)
- 리드 등록, 파이프라인 칸반 보드
- 상태 전환: 신규 → 상담중 → 견적발송 → 계약체결 / 실주
- 리드 상세에서 견적서 작성 바로 연결

### 2. 견적서 작성
- **독성시험** (`/quotations/new`): V2 엔진, 9개 모드, OECD 가이드라인
- **효력시험** (`/efficacy-quotations/new`): 시험 설계 다이어그램 포함
- **임상병리** (`/clinical-pathology/quotations/new`)

### 3. 계약 관리 (`/contracts`)
- 견적서 → 계약서 전환
- 계약 상태, 결제 일정 관리

### 4. 시험 관리 (`/studies`) ← **v1.4.0 신규**
- 시험 목록: 검색, 상태 필터, 페이지네이션
- 시험 상세: 상태 스테퍼 (접수 → 준비 → 진행 → 분석 → 보고서 → 완료)
- 대시보드: 가동률, 지연 시험, 보고서 현황, 캘린더

## 주요 기능

| 기능 | 경로 | 설명 |
|------|------|------|
| 대시보드 | `/dashboard` | 개인/회사 통계, 빠른 링크 |
| 고객사 관리 | `/customers` | CRM 수준 고객 관리, 등급, 태그 |
| 파이프라인 | `/pipeline` | 칸반 보드 드래그앤드롭 |
| 매출 대시보드 | `/sales` | 매출 분석, 전환율, 성과 |
| 통합 검색 | `/search` | 전체 데이터 검색 |
| 캘린더 | `/calendar` | 일정 관리 |
| 통계/리포트 | `/reports` | 견적 통계 |
| 설정 | `/settings` | 사용자 설정, 코드 설정 |
| 관리자 패널 | `/admin` | 사용자 관리, 백업, 자동화 |

## 알려진 제한사항

1. **Render 무료 플랜**: 15분 비활동 시 서버 슬립 → 첫 요청 시 10~30초 대기 (자동 재시도 적용됨)
2. **PWA**: 모바일 홈 화면 추가 가능하나 오프라인 모드는 제한적
3. **데이터**: 시험항목 가격은 참고용, 실제 적용 전 검증 필요

## 피드백 요청사항

베타 테스트 중 다음 사항에 대한 피드백을 부탁드립니다:

1. **워크플로우**: 리드 → 견적 → 계약 → 시험 흐름이 자연스러운지
2. **데이터 정확성**: 시험항목, 가격, 계산 결과가 맞는지
3. **사용성**: 자주 쓰는 기능에 접근이 편한지
4. **모바일**: 모바일에서 주요 기능이 잘 동작하는지
5. **오류**: 에러 발생 시 화면 캡처와 함께 알려주세요
