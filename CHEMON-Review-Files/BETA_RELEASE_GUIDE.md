## 시스템 개요

CHEMON 견적관리 시스템은 비임상 시험 견적서 작성부터 계약, 시험 관리까지의 전체 업무 프로세스를 지원하는 통합 관리 시스템입니다.

### 시스템 구성

- **프론트엔드**: Vercel 배포 (Next.js)
- **백엔드**: Render 배포 (Express.js + PostgreSQL)
- **다중 사용자 지원**: 모든 데이터는 서버에 저장되어 팀원 간 공유됩니다

## 주요 기능

### 1. 대시보드 (`/dashboard`)

- 오늘의 견적/계약 현황 요약
- 최근 견적서 목록
- 월별 추이 차트
- Modality별 분포 차트
- 빠른 작업 바로가기

### 2. 독성시험 견적서 (`/quotations`)

#### 견적서 목록

- 전체 견적서 조회 및 검색
- 상태별 필터링 (작성중, 발송완료, 계약완료 등)
- 견적서 복사, 삭제 기능

#### 견적서 작성 (`/quotations/new`)

**5단계 마법사 방식:**

1. **기본정보**: 고객사, 의뢰인, 프로젝트명, 유효기간
2. **Modality 선택**: 의약품 유형 선택 (저분자, 항체, 세포치료제 등)
3. **시험항목 선택**: 카테고리별 시험 선택, 수량/기간 설정
4. **금액 계산**: 자동 계산, 할인율 적용, 부가세 설정
5. **미리보기**: 최종 확인 및 PDF 다운로드

#### 워크플로우 견적 (`/quotations/workflow`)

- 단계별 가이드 방식의 견적 작성
- Modality → 프로젝트 유형 → 시험 선택 → 검토

#### 견적서 상세 (`/quotations/[id]`)

- 견적서 상세 정보 조회
- PDF 다운로드
- 수정/복사/삭제

### 3. 약효시험 견적서 (`/efficacy-quotations`)

#### 견적서 목록

- 약효시험 전용 견적서 관리
- 유형별 필터 (In-vivo, In-vitro, Ex-vivo)

#### 견적서 작성 (`/efficacy-quotations/new`)

**6단계 마법사 방식:**

1. **기본정보**: 고객사, 의뢰인, 프로젝트명
2. **모델 선택**: 질환 모델 선택 (암, 염증, 대사질환 등)
3. **항목 구성**: 시험 항목 추가 및 설정
4. **시험 설계**: 군 구성, 투여 방법, 기간 설정
5. **금액 계산**: 자동 계산 및 조정
6. **미리보기**: 최종 확인 및 PDF 다운로드

### 4. 리드 관리 (`/leads`)

- 잠재 고객 문의 관리
- 리드 상태 추적 (신규, 상담중, 견적발송, 계약완료, 보류, 실패)
- 리드 → 견적서 → 계약 전환 추적

#### 리드 등록 (`/leads/new`)

- 고객사 정보, 문의 내용, 예상 금액
- 담당자 배정, 우선순위 설정

#### 리드 상세 (`/leads/[id]`)

- 상세 정보 조회 및 수정
- 상태 변경 이력
- 관련 견적서/계약 연결

### 5. 고객 관리 (`/customers`)

#### 고객 목록

- 고객사 검색 및 필터링
- 등급별 관리 (VIP, 일반, 신규)

#### 고객 상세 (`/customers/[id]`)

**탭 구성:**

- **개요**: 기본 정보, 진행 현황 요약
- **의뢰인**: 고객사 담당자 관리
- **미팅기록**: 상담/미팅 이력 관리
- **시험접수**: 시험 접수 현황
- **청구일정**: 청구 스케줄 관리
- **타임라인**: 전체 활동 이력

#### 진행 워크플로우

- 9단계 파이프라인 진행 상태 관리
- 단계별 체크리스트
- 자동 알림 (D-Day 기반)

### 6. 계약 관리 (`/contracts`)

#### 계약 목록

- 전체 계약 조회
- 상태별 필터 (작성중, 검토중, 체결완료 등)

#### 계약 작성 (`/contract/new`)

- 견적서 기반 계약서 생성
- 계약 조건 설정
- 청구 일정 설정

#### 계약 상세 (`/contracts/[id]`)

- 계약 정보 조회
- 계약서 PDF 다운로드
- 변경계약서 생성

### 7. 시험 관리 (`/studies`)

- 진행 중인 시험 현황
- 시험 상태 추적
- 시험번호 관리

### 8. 상담기록 (`/consultations`)

- 고객 상담 기록 관리
- 상담기록서 PDF 생성

### 9. 파이프라인 (`/pipeline`)

- 칸반 보드 형태의 영업 파이프라인
- 드래그 앤 드롭으로 단계 이동
- 단계별 금액 합계

### 10. 일정 관리 (`/calendar`)

- 월간 캘린더 뷰
- 일정 등록/수정/삭제
- 시험 일정, 미팅, 마감일 등 관리
- 고객별 일정 자동 연동

### 11. 패키지 관리 (`/packages`)

- 시험 패키지 템플릿 관리
- 자주 사용하는 시험 조합 저장
- 패키지 기반 빠른 견적 작성

### 12. 리포트 (`/reports`)

- 기간별 실적 리포트
- 담당자별 실적
- 고객사별 분석

### 13. 영업현황 (`/sales`)

- 개인 영업 실적 대시보드
- 목표 대비 달성률
- 월별 추이

### 14. 계산기 (`/calculators`)

비임상 시험 관련 전문 계산기 모음:

| 계산기          | 경로                           | 설명                    |
| --------------- | ------------------------------ | ----------------------- |
| 희석 계산기     | `/calculators/dilution`        | 용액 희석 계산          |
| 용량 환산       | `/calculators/dose-conversion` | 종간 용량 환산          |
| 투여량 계산     | `/calculators/dosing`          | 투여량 및 투여액량 계산 |
| 노출역 계산     | `/calculators/exposure-margin` | 안전역 계산             |
| MRSD 계산       | `/calculators/mrsd`            | 최대권장시작용량 계산   |
| 시험물질 소요량 | `/calculators/test-material`   | 시험물질 필요량 계산    |
| TK 파라미터     | `/calculators/tk-parameters`   | 독성동태 파라미터 계산  |

### 15. 설정 (`/settings`)

#### 프로필 설정

- 개인 정보 수정
- 비밀번호 변경
- 사용자 코드 설정

#### 회사 정보

- 회사 기본 정보 설정
- 견적서/계약서에 표시될 정보

#### 마스터데이터 관리

- 시험항목 관리
- 가격 관리
- 패키지 템플릿 관리

#### 약효시험 설정 (`/settings/efficacy`)

- 약효시험 모델 관리
- 항목 관리

## 관리자 기능

관리자 권한이 있는 계정만 접근 가능합니다.  
**접근 경로**: 우측 상단 프로필 → 관리자 패널

### 1. 관리자 대시보드 (`/admin`)

- 시스템 전체 현황
- 사용자 통계
- 최근 활동 로그

### 2. 사용자 관리 (`/admin/users`)

- 전체 사용자 목록
- 사용자 상태 관리 (활성/비활성/잠금)
- 권한 변경 (일반 사용자 ↔ 관리자)
- 비밀번호 초기화
- 매출 조회 권한 부여

### 3. 매출 통계 (`/admin/sales`)

- 전체 매출 현황 (관리자 전용)
- 기간별 필터링
- 담당자별 실적 비교
- 월별 추이
- 견적→계약 전환율

### 4. 공지사항 관리 (`/admin/announcements`)

- 공지사항 작성/수정/삭제
- 우선순위 설정 (긴급, 높음, 보통, 낮음)
- 게시 기간 설정
- 조회 통계

### 5. 활동 로그 (`/admin/logs`)

- 전체 사용자 활동 기록
- 필터링 (사용자, 액션, 기간)
- 보안 감사용

### 6. 시스템 설정 (`/admin/settings`)

- 회원가입 허용 여부
- 세션 타임아웃 설정
- 로그인 시도 제한
- 계정 잠금 시간
- 백업 관리

## 피드백 요청사항

베타 테스트 기간 동안 다음 사항에 대한 피드백을 부탁드립니다:

### 1. 기능 관련

- [ ] 필요하지만 없는 기능이 있나요?
- [ ] 불필요하거나 혼란스러운 기능이 있나요?
- [ ] 기능의 동작이 예상과 다른 경우가 있나요?

### 2. 사용성 관련

- [ ] 메뉴 구조가 직관적인가요?
- [ ] 화면 이동이 자연스러운가요?
- [ ] 버튼/아이콘의 의미가 명확한가요?

### 3. 데이터 관련

- [ ] 마스터데이터(시험항목, 가격)에 누락된 항목이 있나요?
- [ ] 잘못된 데이터가 있나요?
- [ ] 추가되어야 할 카테고리가 있나요?

### 4. 성능 관련

- [ ] 느리게 느껴지는 화면이 있나요?
- [ ] 오류가 발생하는 경우가 있나요?

### 5. 기타

- [ ] 개선되었으면 하는 점
- [ ] 좋았던 점

## 알려진 제한사항

1. **마스터데이터**: 시험항목 및 가격 데이터는 참고용이며, 실제 업무 적용 전 검증 필요

3. **모바일 최적화**: 데스크톱 환경에 최적화되어 있음
